## how to run
```
cmake build/ # execute only first time or when u add new source file
./run.sh
```
and open output/dashboard.html or check raw data in output.csv.

if u need help:
```
./help.sh
```

## input file format
[number of particles] [radius of particles space]  
[mass0] [x0] [y0] [z0]  
[mass1] [x1] [y1] [z1]  
.  
.  
.  
[mass n - 1] [x n - 1] [y n - 1] [z n - 1]
